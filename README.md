# Term Filter 2

An expanded and extremely customizable rewrite of the in-game term filter, this allows players to mask/hide/modify messages containing specific words. I felt like the vanilla term filter was missing a lot of options, so I added them.

https://github.com/user-attachments/assets/29cb7022-e5ff-448d-b4bb-e6c56e53fe88

With each filter (there is no limit to how many you can have), you can customize each of the following:

* Which chat channel(s) you want it to filter, or all of them!
* Which player(s) you want it to filter, or all of them!
* Whether to hide the message completely (and log it in the /xllog) or show a red "[Filtered] Message could not be displayed due to Term Filter 1."
* Whether to (instead of hiding) replace *only the phrase* in text with another of your choice (or multiple ones it'll pick at random) and bold it so you know it's been replaced.
